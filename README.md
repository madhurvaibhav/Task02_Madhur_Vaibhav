## Student Grade Calculator

A Java console-based application that calculates a student’s total marks, percentage, and grade based on marks entered for multiple subjects.

## Features

* Multiple subject marks input
* Total marks calculation
* Percentage calculation
* Grade assignment
* Input validation

## Technologies Used

* Java
* VS Code
* GitHub

## Learning Outcomes

This project helped me understand loops, conditional statements, user input handling, and basic Java programming concepts.
